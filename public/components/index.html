Brand Design System, powered by tokens.

Change a few values in brand-tokens.css and your slide decks, landing pages, and email layouts update automatically.

View slide layout View email layout

How it works

  1. Bootstrap handles layout and basic components.
  2. Tokens in brand-tokens.css theme Bootstrap.
  3. LLMs output HTML using only Bootstrap classes.

Tokens

Colors, radii, typography, and shadows defined with CSS variables. Non-coders only edit this single file.

Templates

LLM-generated Bootstrap pages for slides, web views, and email layouts, all styled via tokens.

Experiences

Decks, campaigns, and flows made by stacking templates into complete stories.
